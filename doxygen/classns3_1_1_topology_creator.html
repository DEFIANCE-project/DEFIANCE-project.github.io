Detailed Description

This class helps to create realistic scenarios with base stations from OSM and moving user equipment from SUMO.

It provides some default scenarios for you to use. You can also create your scenario by specifying a specific location center and size. Another option is to specify the desired number of base stations and let the class find a pseudo-random location for you (for reproducibility).

Detailed Description

This application is installed on a node on which the RL agent is to be placed in the network.

Before the AgentApplication is started Setup() should be called to initiate necessary configurations. In order to receive observations or rewards from other applications, the AgentApplication needs to add the according interfaces with AddObservationInterface(), AddRewardInterface(), AddActionInterface() or AddAgentInterface(). When an observation or a reward is received, the AgentApplication stores it in a data structure and calls OnRecvObs() resp. OnRecvReward() with the ID of the interface the observation or reward was received from. The AgentApplication needs to implement OnRecvObs() and OnRecvReward() to handle received observations and rewards.
